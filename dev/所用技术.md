# 项目技术栈

**以下技术栈都是暂定，具体用什么日后由前端同学 后端同学自行讨论。**

## 前端

- 框架技术：Vue3
- 基础语言：JavaScript
- 类型推断：TypeScript
- 界面组件：Element-Plus
- 路由管理：Vue-Router v4.x
- 状态管理:  Pinia
- 网络请求：Axios
- 即时通讯：HTML5-Websocket
- 依赖管理：Yarn
- 打包工具：WebPack

## 后端

- 基础语言：Java
- 基础框架：Spring-Boot
- 安全框架：自己做
- 框架集成：Mybatis
    Mybatis-Plus
    LomBok
    Jedis
    .....
- 令牌通信：JWT
- 数据存储：Mysql
- 数据缓存：Redis
- 文件服务：OSS服务
- 邮件服务：SpringMail
- 即时通讯：Spring-WebSocket
- 消息队列：RabbitMQ
- 依赖管理：Maven
- 反向代理：Nginx
- 服务容器：Tomcat
- 操作系统：Ubuntu20.02LTS
